## Required File

You need to add the following audio file to this directory:

### `last-christmas.mp3`

**Song:** Last Christmas by Wham!

**How to get the file:**

1. **Option 1: Download from YouTube**
   - Go to: https://www.youtube.com/watch?v=E8gmARGvPlI
   - Use a YouTube to MP3 converter (e.g., https://ytmp3.cc/)
   - Download as MP3
   - Rename to `last-christmas.mp3`
   - Place in this `audio/` folder

2. **Option 2: Use a free music library**
   - Search for "Last Christmas instrumental" or royalty-free version
   - Download as MP3
   - Rename to `last-christmas.mp3`
   - Place in this `audio/` folder

3. **Option 3: Use your own MP3 file**
   - If you have the song in MP3 format
   - Rename to `last-christmas.mp3`
   - Place in this `audio/` folder
